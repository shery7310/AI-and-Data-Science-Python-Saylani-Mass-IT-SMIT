#### Adding a Todo

To add a todo, Select `POST` from POSTMAN, hit/enter this url:
`http://localhost:8000/add_todo`, then select Headers and Set key as Content-Type and it's value as application/json. 
Then Select Body and pass a sample todo i.e.

```json
{
"userId": "user-009",
"title": "Sleep On Time",
"completed": "True",
"priority": "high",
"category": "work",
"notes": "Hasbun Allah Hu Wa Namul Wakeel"
}
```
